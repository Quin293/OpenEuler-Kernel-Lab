编写字符设备程序，使之满足以下功能：

i. 安装设备后从设备中读出字符串；

 ii.  设备支持每次写入字符不超过1024个，超过部分被丢弃，并且能够保存最近一次修改前的rwbuf里的内容。

 iii. 设备支持系统调用ioctl(int d, int req,…),共支持设置三种模式：

     a. 清除设备中写入的字符串; 

     b. 从设备中读时，读出的是最近一次修改前的内容。 

     c. 从设备中读时，读出的是最新内容

 iv. 设备关闭前不能被多次打开；

 v. 自己编写测试程序，验证以上功能


命令行切换到文件所在路径下，输入命令：
sudo make 编译
sudo insmod rwbuf.ko 安装设备
sudo chmod 666 /dev/rwbuf 给予设备权限

再运行test文件里的测试程序即可
